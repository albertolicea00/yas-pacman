# Design System - Yet Another Store for Pacman

This document outlines the visual identity and styling guidelines for **Yet Another Store for Pacman**.

## Color Palette

The design is built around a modern, premium Dark Theme featuring a deep store base color combined with a signature Orange accent.

### Color Tokens

- **Store Base Color (Dark Gray)**:
  - Hex: `#1E1E2E`
  - HSL: `hsl(240, 21%, 15%)`
- **Accent Color (Orange)**:
  - Hex: `#FF5722`
  - HSL: `hsl(14, 100%, 57%)`
- **Subtle Highlight**:
  - Hex: `#FF57221A` (10% opacity)
- **Text Primary**:
  - Hex: `#F8F8F2`
- **Text Secondary**:
  - Hex: `#A9B1D6`

## Visual Assets & Logo Concept

- **Base Structure**: Dark gray store icon outline or 3D block representing the storefront.
- **Accent Elements**: Highlights, borders, or light beams in **Orange**.
- **Inside the Store**: Stacked package cubes representing the packages managed by Pacman.
- **App Tag**: PACMAN

## Typography

- **Headings**: Outfit or Inter (sans-serif)
- **Body Text**: Inter (sans-serif)
- **Monospace (for CLI output/commands)**: Fira Code or JetBrains Mono
